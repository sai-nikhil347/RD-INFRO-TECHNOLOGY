# Task 7 - Monitor Cloud Resources using AWS CloudWatch

## Objective
Monitor AWS EC2 resources using Amazon CloudWatch and configure an alarm for high CPU utilization.

## Services Used
- Amazon EC2
- Amazon CloudWatch
- Amazon SNS

## Steps Performed

1. Launched an EC2 instance.
2. Opened Amazon CloudWatch.
3. Navigated to Metrics → EC2 → Per-Instance Metrics.
4. Selected CPUUtilization metric.
5. Viewed CPU usage graph.
6. Created a CloudWatch Alarm.
7. Set threshold:
   - CPUUtilization > 70%
8. Configured SNS email notification.
9. Successfully created the alarm.

## Outcome

Successfully monitored EC2 CPU utilization and configured an automatic CloudWatch Alarm for high CPU usage.
